Phone Audio Recorder를 다른 컴퓨터에서 쓰는 방법

가장 쉬운 방법:
1. GitHub에서 코드를 받습니다.
2. 폴더 안에서 Phone Audio Recorder.bat 를 더블클릭합니다.
3. 이어 녹음 최종 저장이 안 되면 Install FFmpeg.bat 를 한 번 실행합니다.

중요:
- Phone Audio Recorder.bat 파일만 옮기면 안 됩니다.
- tools 폴더가 반드시 같이 있어야 합니다.
- 폴더 전체를 통째로 옮겨야 합니다.
- GitHub에는 용량 제한 때문에 tools\ffmpeg.exe 본체는 제외되어 있습니다.
- 여러 조각을 하나로 합치려면 FFmpeg가 필요합니다.

노트북에서 처음 한 번 해야 할 것:
1. 갤럭시 개발자 옵션에서 USB 디버깅 ON
2. USB 케이블로 노트북에 연결
3. 폰에 USB 디버깅 허용 창이 뜨면 "항상 허용" 체크 후 허용
4. 프로그램에서 폰 연결 확인

추천 설정:
- 저장 형식: M4A / AAC 320K
- 오디오 캡처 방식: output - 기본값, 전체 폰 소리
- 첫 소리 이후 무음이면 일시정지: 켜기
- 무음 시간: 원하는 초로 설정

녹음 흐름:
1. 녹음 시작
2. 설정한 시간만큼 무음이면 자동 일시정지
3. 더 녹음하려면 이어 녹음
4. 끝났으면 최종 저장
5. 최종 저장 파일 하나가 저장 폴더에 생성됩니다.

안 될 때:
- 폰 연결 확인에서 unauthorized가 나오면 폰에서 USB 디버깅 허용을 눌러야 합니다.
- 허용 창이 안 뜨면 개발자 옵션에서 USB 디버깅 권한 취소를 누르고 케이블을 다시 연결하세요.
- 무음 일시정지가 너무 빨리 작동하면 체크를 끄고 수동으로 일시정지를 누르세요.
