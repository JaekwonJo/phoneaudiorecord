Phone Audio Recorder

실행:
1. Phone Audio Recorder.bat 를 더블클릭하세요.
2. Windows 보안 창이 뜨면 허용하세요.

사용 순서:
1. 갤럭시에서 USB 디버깅을 켭니다.
2. USB 케이블로 PC에 연결합니다.
3. 폰에 "USB 디버깅 허용" 창이 뜨면 허용을 누릅니다.
4. 프로그램에서 "폰 연결 확인"을 누릅니다.
5. 저장 형식을 고릅니다.
   - 추천: M4A / AAC 192K
   - 더 좋은 음질: M4A / AAC 320K 또는 FLAC
   - 편집용: WAV
6. "녹음 시작"을 누릅니다.
7. 폰에서 Gemini 음성을 재생합니다.
8. 무음이 10초 지속되면 자동으로 일시정지됩니다.
9. 더 녹음하려면 "이어 녹음"을 누릅니다.
10. 끝났으면 "최종 저장"을 누릅니다.
11. "저장 폴더 열기"를 눌러 파일을 확인합니다.

참고:
- 기본 저장 위치는 음악\Phone Audio Recorder 입니다.
- Windows 보안/방화벽 창이 뜨면 개인 네트워크 허용을 누르면 됩니다.
- output 방식이 안 되면 오디오 캡처 방식을 playback으로 바꿔 테스트하세요.
- 다른 컴퓨터에서 쓰려면 이 폴더 전체를 복사하세요. tools 폴더가 같이 있어야 합니다.
- 무음 일시정지는 PC에서 실제로 들리는 소리를 기준으로 감지합니다.
- 폰 미디어 볼륨은 녹음 크기이고, PC 볼륨은 듣기 크기입니다.
- 저장 폴더, 저장 형식, 캡처 방식, 무음 일시정지 설정은 자동 저장됩니다.
- 여러 번 이어 녹음한 파일은 FFmpeg로 최종 저장할 때 하나로 합쳐집니다.
- GitHub에서 받은 경우 여러 조각 합치기가 안 되면 Install FFmpeg.bat 를 한 번 실행하세요.
